🧪 留言系統功能測試

📋 測試概述

這個測試將驗證管理員留言系統的完整功能，包括：

💬 管理員功能

  • 管理員控制台新增留言管理選項
  • 選擇目標用戶
  • 發送留言
  • 即時通知

🔔 用戶功能

  • 右上角顯示留言通知
  • 點擊查看留言內容
  • 標記為已讀
  • 多條留言管理

🚀 測試步驟

步驟 1: 管理員發送留言

  1. 使用管理員帳號登入系統
  2. 進入"👑 管理員控制台"
  3. 點擊新增的"💬 留言管理"功能
  4. 選擇一個用戶（如PM或AM）
  5. 輸入留言內容
  6. 點擊發送

步驟 2: 用戶接收留言

  1. 登出管理員帳號
  2. 使用被發送留言的用戶帳號登入
  3. 檢查右上角是否出現留言通知
  4. 點擊通知查看留言內容
  5. 點擊"已讀"按鈕

步驟 3: 多條留言測試

  1. 重複步驟1，發送多條留言給同一用戶
  2. 驗證用戶能看到留言數量提示
  3. 逐一查看和標記已讀

🔍 預期結果

  • ✅ 管理員控制台顯示"💬 留言管理"選項
  • ✅ 留言管理頁面功能完整
  • ✅ 用戶能即時收到留言通知
  • ✅ 留言通知顯示在右上角美觀位置
  • ✅ 點擊通知能查看完整留言內容
  • ✅ 標記已讀功能正常
  • ✅ 多條留言管理正常
