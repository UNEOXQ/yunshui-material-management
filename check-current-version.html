4. 建議的解決方案

如果您正在使用 3000 端口，可能的問題：
  • 您可能在使用構建版本（dist 文件夾）而不是開發版本
  • 瀏覽器緩存了舊的 JavaScript 文件
  • 需要重新構建前端代碼
建議的解決步驟：
  1. 清除瀏覽器緩存（Ctrl + Shift + R）
  2. 重新構建前端：cd frontend && npm run build
  3. 或者使用開發模式：cd frontend && npm run dev
